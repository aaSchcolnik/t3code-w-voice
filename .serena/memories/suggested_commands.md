# Suggested Commands

- Install deps: `vp i`
- Run full dev stack: `vp run dev`
- Run server dev: `vp run dev:server`
- Run web dev: `vp run dev:web`
- Run desktop dev: `vp run dev:desktop`
- Build: `vp run build`
- Typecheck: `vp run typecheck`
- Lint: `vp run lint`
- Format: `vp run fmt`
- Format check: `vp run fmt:check`
- Test all package scripts: `vp test`
- Run package `test` scripts specifically: `vp run test`
- Required completion checks: `vp check` and `vp run typecheck`
- If changing native mobile code, also run: `vp run lint:mobile`
- Search files/text: prefer `rg --files` and `rg`.
- Git status: `git status --short`.
